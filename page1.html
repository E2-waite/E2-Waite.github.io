My Projects

Throughout my time at university I developed a number of projects to specifications, making use of a variety of different technologies.

Below are some of these projects.

I hope you like what you see!

Procedural Sokoban Generator
Procedural generation system capable of generating Unique Sokoban puzzles as part of a larger maze structure created for my final university dissertation project.

C#, Unity

DirectX11 Foliage Model Generator
System for generating 3D models of foliage developed with DirectX11
The outputted models can be modified by adjusting parameters and the resulting models can be export to the .OBJ file type.

C++, DirectX11

Back In My Day (VR Experience)
A Virtual Reality experience that takes a look at Bristol's underground music scene developed as part of a group in Unity for the Oculus Quest.

C#, Unity, Oculus Quest

Don't Walk By
Polished game created as part of a larger cohort, made up of smaller teams with each team taking on individual responsibilities and game mechanics.

C#, Unity

DirectX11 Open World System
Open world system developed in C++ using DirectX11. Streams geometry data as a player moves through the environment, reducing the amount of data stored in memory.
The system makes use of binary serialization for the streaming of data.
A third person controller was also developed in order to navigate throughout the environment.

C++, DirectX11

Noisy Dungeons
Dungeon crawler game developed in Unity which heavily makes use of sound.
The audio is controlled using FMod - allowing for the audio to be changed dynamically.
Developed in Unity as part of a group.

C#, Unity, FMod

Code-Crunch
Local multiplayer turn-based racing game developed for android in Unity.
The objective is to race to the end by dragging tiles that drop from the top of the screen in order to make your character move.
Developed in Unity as part of a group.

C#, Unity

Ark-Devolved
Survival game based on Ark Survival Evolved in the theme of the Atari 2600.
Developed in Unity as part of a group.

C#, Unity

Industrial-Lives
Networked multiplayer turn-based strategy game programmed in C++ in which the players take turns to take over the map and sabotage each other.

C++

RoboLab
Robot lab management simulator developed in C++ as part of a group for the Low Level Programming second-year module for BSc Games Technology.

C++

A Heart-Lifting Game
Developed as part of a group in 48 hours for the Global Game Jam 2020.
The objective is to attempt to reconstruct your spaceship then avoid obstacles to return to your planet.

C#, Unity

Haunted House
Text adventure that I have ported from the BASIC coding language into C++ for the Low Level Programming second-year module for BSc Games Technology.

C++
