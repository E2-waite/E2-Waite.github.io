Noisy Dungeons

GitHub

Description

Noisy Dungeons is a single player dungeon crawler game that was developed as part of a team for the Commercial Games Development third-year module of BSc Games Technology. 

FMOD

Through the use of FMOD 

Enemies

I was responsible for the development of audio responsive enemies that chase and attack the player when the player's footsteps are heard. 
The enemies navigate through the dungeon using a NavMesh which allows them create a path towards a desired location and avoid obstacles in the way.

Pyro

The pyro is a ranged enemy which fires a fireball projectile when the player is heard and is in line of sight. 

Spider

Traps

I was tasked with the creation of a number of traps which are scattered throughout the level which the player must navigate around.
As the player is unable to attack the enemies throughout the dungeon - the only way that the player is able to defeat the enemies is by leading them into the traps through the manipulation of their audio responsive mechanics.
