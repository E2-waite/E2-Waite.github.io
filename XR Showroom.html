Cobra XR Showroom

Description

The Cobra XR Showroom is a tool used to demonstrate Cobra Simulation's immersive domes in mixed reality through the use of the Oculus Quest 1/2's passthrough functionality. This allows for the showcasing of the company's products at a 1:1 scale in a real-world environment without the need to transport the physical product.
I played a role in the development of this tool by implementing the snapping functionality for combining dome sections to demonstrate their modular nature. I was also tasked with the implementation of some of the tool's GUI.
This tool was developed in Unity with the Oculus Integration SDK.

My Contributions

Dome Part Snapping

I was responsible for implementing the functionality for snapping dome parts onto domes to demonstrate how each part is assembled and to show the different possible configurations. This also involved the implementation of a serialization system for saving different dome configurations to allow for faster deployment.

Menu Functionality

I was tasked with the implementation of the main menu functionality and the part snapping menu.
